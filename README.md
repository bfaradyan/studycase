### tech stack used in this project
- laravel 11.0
- bootsrap 5.0

## preview/run

To Preview this project run

```bash
git clone https://github.com/fauzymadani/laravelCrud.git
```

install dependencie's
```bash
composer install
```

```bash
  php artisan serve
```
